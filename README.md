📒 Contact Book System (Python Console Application)

📌 Overview
The Contact Book System is a Python-based console application designed to efficiently manage personal or basic contact information. It allows users to perform essential CRUD operations (Create, Read, Update, Delete) on contact records using a menu-driven interface. This project demonstrates the use of core Python concepts such as lists, dictionaries, loops, functions, and conditional statements.

 🎯 Key Features
- Add new contacts with name and phone number  
- Search contacts by name  
- Delete existing contacts  
- View all stored contacts  
- Exit the application safely  

 🧠 Concepts Used
- Python Lists & Dictionaries  
- Functions for modular programming  
- Looping (while & for loops)  
- Conditional statements (if-elif-else)  
- User input handling  

⚙️ How It Works
The system runs in a continuous loop and displays a menu to the user. Based on the user’s input, it performs the selected operation such as adding a contact, searching for a contact, deleting a contact, or displaying all contacts. The program continues running until the user chooses the exit option.
